# Planning Prompt
Design a schema-first backend implementation plan for Studio control plane changes.

## Inputs
- Requested behavior and entities
- Existing contracts under apps/api/src/modules/** and packages/**
- AGENTS.md constraints

## Output format
1. Contract changes
2. Entity/model changes
3. Runtime orchestration changes
4. Propagation and loop-prevention strategy
5. Touched files list
6. Test plan
7. Docs updates required
